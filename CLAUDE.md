## Project Overview

junit-rtgen is a Rust CLI tool that converts JUnit XML format files to ParallelTests::RSpec::RuntimeLogger format. This conversion helps optimize parallel test execution in Ruby/RSpec projects.

## Architecture

This is a command-line application for converting test report formats. The main components will likely include:

1. **XML Parser** - For reading JUnit XML format files
2. **Data Model** - Internal representation of test results
3. **Formatter** - For outputting ParallelTests::RSpec::RuntimeLogger format
4. **CLI Interface** - Command-line argument parsing and user interaction

The tool's purpose is to enable Ruby/RSpec projects using ParallelTests to leverage JUnit XML reports (commonly generated by various CI systems) for optimizing test distribution across parallel workers.
